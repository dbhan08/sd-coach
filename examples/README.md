# Examples

Synthetic example sessions, one per mode. **These are illustrative, not transcripts of real runs** — they exist to show what good output from each mode looks like. Real sessions you generate locally land in `sessions/` (gitignored) and stay private.

When you generate a real session you're proud of, you can promote it here by copying the folder out of `sessions/`, scrubbing anything personal, and adding a note at the top: "Real run, <date>."
